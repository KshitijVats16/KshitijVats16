### 🚀 What I work on

I build systems that go beyond notebooks — hybrid retrieval pipelines, LLM-orchestrated agents, and end-to-end deployments that actually run in production, not just locally.

- 🔍 **Retrieval-Augmented Generation (RAG):** hybrid dense + sparse retrieval (FAISS + BM25), semantic search, re-ranking
- 🤖 **LLM Agents:** multi-turn conversational systems, prompt engineering, guardrails (off-topic + prompt-injection detection)
- 🐳 **Deployment:** Docker containerization, cloud hosting (Hugging Face Spaces, Render), CI-friendly API design with FastAPI
- 📊 **Applied ML:** classical ML (TF-IDF, LinearSVC) through modern embedding-based systems
